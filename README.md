El presente proyecto consiste en el desarrollo de una aplicación móvil diseñada para optimizar los procesos de difusión informativa dentro de una comunidad académica. El Sistema de Comunicación Institucional Móvil (SCIM) centraliza avisos administrativos, eventos académicos y alertas de contingencia (como suspensiones de actividades) para asegurar una comunicación efectiva y oportuna entre la institución y sus integrantes.

Objetivos Académicos:

Optimización de la Gestión Informativa: Sistematizar el flujo de avisos institucionales para reducir el ruido informativo y la desinformación.
Inmediatez en Alertas de Contingencia: Implementar protocolos de comunicación en tiempo real para eventos imprevistos, tales como la suspensión de clases o cambios en el calendario académico.
Fomento de la Participación Institucional: Incrementar el compromiso de la comunidad mediante el acceso centralizado a noticias y calendarios escolares.

Funcionalidades del Sistema
📡 Difusión por Segmentos: Envío de notificaciones diferenciadas para alumnos, docentes y personal administrativo.
⚠️ Alertas de Prioridad: Notificaciones de alta visibilidad para suspensiones de clases por causas climáticas, sanitarias o administrativas.
